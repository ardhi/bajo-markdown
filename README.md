Markdown support for [Bajo](https://github.com/ardhi/bajo).

## Installation

Goto your ```<bajo-base-dir>``` and type:

```bash
$ npm install bajo-markdown
```

Now open your ```<bajo-data-dir>/config/.plugins``` and put ```bajo-markdown``` in it
